# Ramisa Maliyat's Anime Portfolio Website

Your professional portfolio is ready with all requested features:

## Quick Start
1. Open `index.html` in browser (double-click or `start index.html`).
2. Full responsive across devices.
3. No installation needed (CDNs for Bootstrap/AOS/Fonts).

## Features
- **Anime Theme**: Pink/purple/blue gradients, heart cursor trail, bubble + **rain effects** (full hero backgrounds).
- **Animations**: AOS scroll-ins, card flips/glows (radiant borders), hover transforms, page fade-flips.
- **Pages**: Home, About (expanded), Projects (filterable), Skills (20+ bars), Contact (validation).
- **Responsive**: Mobile/tablet/desktop optimized.

## Customization
- Replace images/emails/links.
- Add resume.pdf to root.
- Edit content in HTML files.

## Deploy
```
git init
git add .
git commit -m "Portfolio"
gh repo create ramisa-portfolio --public
git push origin main
gh pages main
```

Enjoy! 💖✨🌧️
